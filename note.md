## 架构
account 是管理钱包的

consensu 是关于 mining algorithn的

cmd 是geth客户端的cli

eth 是用于让你的节点和最新的区块保持同步

ethdb是关于数据存储的 使用的是leveldb（key - value存储）

minng 是挖矿

rpc 是管理各种借口的 （有json rpc/websocket api/ ipc api
